## 1. การตั้งค่า LINE แจ้งเตือน (LINE Notification Setup)

เพื่อให้พนักงานสามารถรับการแจ้งเตือนงานใหม่และอัปเดตงานผ่าน LINE ได้:

1. **ขอ LINE User ID:** พนักงานต้องใช้ LINE User ID ของตนเอง (สามารถหาได้จากบอทของระบบหรือใช้เครื่องมือเช่น LINE Developers Console)
2. **บันทึก LINE User ID:**
    - ไปที่เมนู **"โปรไฟล์"** (คลิกที่ชื่อผู้ใช้ด้านขวาบน)
    - ใส่ค่า LINE User ID ในช่อง **"LINE User ID"**
    - กด **"บันทึก"**
3. **ระบบจะเริ่มส่งการแจ้งเตือน:** เมื่อมีการมอบหมายงานใหม่หรืออัปเดตงานที่เกี่ยวข้องกับคุณ ระบบจะส่งข้อความแจ้งเตือนไปยัง LINE โดยอัตโนมัติ

---

## 2. การออกรายงาน PDF (PDF Report Generation)

ระบบรองรับการออกรายงานสรุปข้อมูลในรูปแบบ PDF:

1. ไปที่เมนู **"รายงาน" (Reports)**
2. เลือกประเภทรายงานที่ต้องการ:
    - **รายงานงาน (PDF):** สรุปรายการงานทั้งหมดที่มีในระบบ พร้อมสถานะและความคืบหน้า
    - **รายงานบุคลากร (PDF):** สรุปภาระงานของพนักงานแต่ละคน (งานทั้งหมด, งานที่กำลังทำ, งานที่เสร็จแล้ว, งานที่เกินกำหนด)
3. ระบบจะสร้างไฟล์ PDF และดาวน์โหลดลงเครื่องโดยอัตโนมัติ

## 3. การจัดการภาระงานและสถานะงาน (Task & Workload Management)

1. **แดชบอร์ด (Dashboard):** ดูภาพรวมงานทั้งหมด อัตราการปิดงาน และงานที่ใกล้ครบกำหนด
2. **จัดการงาน (Tasks):** สามารถสร้าง แก้ไข และลบงานได้ (สำหรับแอดมิน) หรือเปลี่ยนสถานะงาน (สำหรับพนักงาน)
3. **ภาระงานทีม (Workload):** ดูการกระจายงานของคนในทีม เพื่อประเมินความสมดุลของภาระงาน

---

## 4. การจัดการพนักงาน (Staff Management)

*เฉพาะแอดมิน (Admin) เท่านั้น:*

1. ไปที่เมนู **"พนักงาน" (Staff)**
2. **เพิ่มพนักงานใหม่:** คลิกปุ่ม "เพิ่มพนักงาน" และระบุข้อมูลพื้นฐาน รวมถึง LINE User ID (หากมี)
3. **แก้ไขข้อมูล:** สามารถแก้ไขชื่อ ตำแหน่ง บทบาท และหน่วยงานของพนักงานได้
4. **รีเซ็ตรหัสผ่าน:** สามารถเปลี่ยนรหัสผ่านให้พนักงานได้หากพนักงานลืมรหัสผ่าน
